# React-Maps

This is a little proyect build using modern technologies like:
- React
- Bootstrap/Reactstrap
- Javascript ES6

And libraries like:
- Amcharts
- Lodash
- React Router
- Aviality Reactstrap Validations 
- And more...

## Instrucctions:

First download this repository or clone it and then:

### NPM
```sh
npm install
```

Wait until npm finish the instalation and then
```sh
npm start
```
And that's it

Or you can just view this page on the web deployed in Heroku: [Here](https://react-maps-ricardo.herokuapp.com/)

## Atention
If the page gives an application error saying **"The page could not be served"** just give it a few seconds and refresh. It's a bug we're working on.
